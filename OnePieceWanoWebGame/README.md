# 🏴‍☠️ One Piece: Wano Arc Adventure

A mobile-optimized HTML5 game featuring the Straw Hat Pirates in their epic Wano Country adventure!

## 🎮 How to Play on iOS

### Option 1: Direct File Access
1. Open Safari on your iOS device
2. Navigate to the game folder
3. Tap on `index.html` to start playing

### Option 2: Local Server (Recommended)
1. The game is running on a local server at: `http://localhost:8000`
2. Open Safari and navigate to that URL
3. For best experience, add to home screen:
   - Tap the Share button in Safari
   - Select "Add to Home Screen"
   - The game will run like a native app!

## 🎯 Game Features

### 🏴‍☠️ Playable Characters
- **Monkey D. Luffy** - Gomu Gomu no Mi powers (unlocked from start)
- **Roronoa Zoro** - Three Sword Style (unlock in Chapter 1)
- **Vinsmoke Sanji** - Black Leg Style (unlock in Chapter 2)
- **Nami** - Clima-Tact weather powers (unlock in Chapter 2)
- More characters unlock as you progress!

### ⚔️ Combat System
- **Basic Attacks**: Tap the sword button (⚔️)
- **Devil Fruit Powers**: Tap the fire button (🔥)
- **Haki Abilities**: Tap the lightning button (⚡)
- **Jump**: Tap the kangaroo button (🦘)

### 🎮 Controls
- **Virtual Joystick**: Move your character around
- **Action Buttons**: Combat abilities on the right side
- **Touch Optimized**: Designed specifically for mobile devices

### 📖 Story Mode
Experience the Wano Arc through 5 epic chapters:
1. **Arrival at Wano** - Meet the samurai and learn about the country
2. **Prison Break** - Escape from Udon Prison with allies
3. **Gathering Allies** - Unite the rebels and samurai
4. **Fire Festival Preparation** - Prepare for the final battle
5. **Onigashima Raid** - Face Kaido and the Beast Pirates!

### 🏆 Enemy Types
- **Beast Pirates** - Basic enemies with clubs and horns
- **Wano Samurai** - Skilled swordsmen with katanas
- **Wano Ninja** - Fast-moving stealth fighters
- **SMILE Gifters** - Devil Fruit users with animal powers
- **Headliners** - Elite Beast Pirates officers
- **Kaido** - The ultimate boss battle!

## 🐛 Debug Commands
Open Safari's developer console and try these:
```javascript
// Unlock all characters
debugGame.unlockAllCharacters();

// Set player health
debugGame.setPlayerHealth(500);

// Spawn enemies
debugGame.spawnEnemy('kaido');

// Enable god mode
debugGame.toggleGodMode();

// Complete current chapter
debugGame.completeChapter(2);
```
